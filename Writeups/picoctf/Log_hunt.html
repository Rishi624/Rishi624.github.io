// Given Challenge

Our server seems to be leaking pieces of a secret flag in its logs. The parts are scattered and sometimes repeated. Can you reconstruct the original flag? Download the logs and figure out the full flag from the fragments.

This is a very easy challenge. First, download the given log files manually or using the wget tool.

wget "https://challenge-files.picoctf.net/c_amiable_citadel/49cec6157142f24a599f4164d5b63322c2494f801390d6f22eb91b3aa592bc66/server.log"

After downloading the file, inspect the file using the cat command:

cat server.log

In the first line, you can see there is some part of the flag written after FLAGPART. We need to sort all lines containing "FLAGPART".

For that, we will be using grep with the cat command:

cat server.log | grep "FLAGPART"

Using this command, you can get the flag by combining all the parts found in the output.
